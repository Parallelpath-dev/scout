Reliability

AI Guardrails

Three hard rules applied to every synthesis. These constrain the model to only what the data supports — no extrapolation, no hallucination.

01

Citation Enforcement

The Analyst may only reference competitors, keywords, and numbers present in that week's collected signals. It cannot introduce facts from training data or memory.

02

Confidence Gating

If fewer than 3 signals exist for a competitor in a given category, the output flags the gap rather than speculating. Thin data produces a quality flag, not a fabricated observation.

03

Schema Enforcement

All output is constrained to a defined JSON schema. Named fields, typed values, bounded arrays. This limits fabrication surface area and ensures the dashboard always has structured data to render.
